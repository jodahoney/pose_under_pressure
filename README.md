# Pose Under Pressure

**Pose Under Pressure: Robustness of Pose-Derived Dense Rewards in Demonstration-Guided Reinforcement Learning**

This project studies how structured pose-estimation errors affect reinforcement learning from pose-derived dense imitation rewards. In a controlled continuous-control simulator, ground-truth keypoints are corrupted with noise models such as Gaussian jitter, occlusion dropout, and systematic bias before being used to compute the reward. The goal is to understand how reward corruption affects PPO learning, reward-task alignment, and final imitation quality.
